HTML5 ★ BoilerStrap

A rock-solid default for HTML5 awesome combined with a toolkit from Twitter designed to kickstart development of webapps and sites.

A Mashup of HTML5 Boilerplate & Twitter's Bootstrap

HTML5 Boilerplate is an amazing starting point for building an HTML/CSS/JS based site. Twitter's Bootstrap makes rapid prototyping just that. This project aims to combine the two while keeping both HTML5 Boilerplate's build scripts functioning and allowing you to use all of Bootstrap's CSS and JS awesomeness. The project strives to leave the HTML5 boilerplate and bootstrap relatively untouched. It's a work in progress, as evidenced by the todo list.

What's next? Look at the How To to learn how to get the repo and start your first project. Already bored? Skip straight to some examples.

How To

Step 1

git clone the project recursively. Bootstrap is included as a submodule.

> git clone --recursive git@github.com:elgreg/html5-boilerstrap.git

View on Github

Step 2

Start editing /index.html, /css/style.css and /js/script.js and /js/plugins.js as you see fit.

Learn more about what to put where from the boilerplate docs.

Step 3

Take advantage of bootstrap's startup css and js for grid systems, typography, media, tables, forms, navigation, alerts, and popovers.

Step 4

Publish! Use the build script from boilerplate to create a publish directory with minified css, js, and more.
